Object metadata is not loaded automatically with the scene and may require additional server requests depending on what data you need.
Generally you will work with either <code>[HierarchicalObjectReference](https://docs.novorender.com/legacy/docs/webgl-api/interfaces/NovoRender.HierarcicalObjectReference)</code> or the full <code>[ObjectData](https://docs.novorender.com/legacy/docs/webgl-api/interfaces/NovoRender.ObjectData)</code> metadata set.
